IDEA快捷键总结

以下为我比较常用的快捷键，一般目前会用到的就是这些了吧

1.CTRL

快捷键 介绍
Ctrl + C / V 粘贴 / 复制
Ctrl + A 全选
Ctrl + S 保存
Ctrl + Z 撤销
Ctrl + X 剪切
Ctrl + / 注释
Ctrl + D 复制当行代码
Ctrl + F / R 查找 / 替换
Ctrl + home / end 跳转到文件开始 / 末尾
Ctrl + P 方法参数提示
Ctrl + H 打开层次（继承）关系
Ctrl + W 递进式选择代码块
Ctrl + B 跳转到方法 / 变量位置
Ctrl + + / - 代码块的展开 / 收拢
Ctrl + 左右方向键 单词间跳转

2.SHIFT

快捷键 介绍
Shift + F6 文件 / 类 / 方法重命名
Shift + Tab 取消缩进
Shift + Enter 开始新的一行

3.ALT

快捷键 介绍
Alt + Enter 提供快速修复
Alt + Insert 代码快速生成
Alt + 方向左右键 打开程序之间快速跳转
Alt + 方向上下键 不同方法间的快速跳转
Alt + / 提示补全代码

4.多个组合键

快捷键 介绍
Ctrl + Shift + F10 运行当前代码
Ctrl + Alt + S 打开设置
Ctrl + Alt + Shift + S 打开项目设置
Ctrl + Shift + 方向上下键 本行 / 选中代码上下移
Ctrl + Shift + / 选中代码注释
Ctrl + Shift + Enter 快速分割代码
Ctrl + Alt + 方向左右键 回到上一个操作的地方
Ctrl + Shift + U 代码大小写替换

5.快捷输入

快捷字符 介绍
main / psvm main方法，程序入口
sout println打印
fori for循环
ifn if (参数 == null)判断

以上都是自带的常用的总结

IDEA设置中可以添加自己的快捷输入，位置：setting – Editor – Live Remplates

在这里就可以进行设置了，可以添加一个MyKeys，然后添加自己的快捷输入

以上便是我们常会用到的IDEA快捷键了，如有更多需求，可参考以下或者网上查阅：

https://blog.csdn.net/qq_38963960/article/details/89552704
