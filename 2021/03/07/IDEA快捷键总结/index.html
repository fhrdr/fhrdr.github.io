IDEA快捷键总结

以下为我比较常用的快捷键，一般目前会用到的就是这些了吧

1.CTRL

快捷键介绍
Ctrl + C / V粘贴 / 复制
Ctrl + A全选
Ctrl + S保存
Ctrl + Z撤销
Ctrl + X剪切
Ctrl + /注释
Ctrl + D复制当行代码
Ctrl + F / R查找 / 替换
Ctrl + home / end跳转到文件开始 / 末尾
Ctrl + P方法参数提示
Ctrl + H打开层次（继承）关系
Ctrl + W递进式选择代码块
Ctrl + B跳转到方法 / 变量位置
Ctrl + + / -代码块的展开 / 收拢
Ctrl + 左右方向键单词间跳转

2.SHIFT

快捷键介绍
Shift + F6文件 / 类 / 方法重命名
Shift + Tab取消缩进
Shift + Enter开始新的一行

3.ALT

快捷键介绍
Alt + Enter提供快速修复
Alt + Insert代码快速生成
Alt + 方向左右键打开程序之间快速跳转
Alt + 方向上下键不同方法间的快速跳转
Alt + /提示补全代码

4.多个组合键

快捷键介绍
Ctrl + Shift + F10运行当前代码
Ctrl + Alt + S打开设置
Ctrl + Alt + Shift + S打开项目设置
Ctrl + Shift + 方向上下键本行 / 选中代码上下移
Ctrl + Shift + /选中代码注释
Ctrl + Shift + Enter快速分割代码
Ctrl + Alt + 方向左右键回到上一个操作的地方
Ctrl + Shift + U代码大小写替换

5.快捷输入

快捷字符介绍
main / psvmmain方法，程序入口
soutprintln打印
forifor循环
ifnif (参数 == null)判断

以上都是自带的常用的总结

IDEA设置中可以添加自己的快捷输入，位置：setting — Editor — Live Remplates

在这里就可以进行设置了，可以添加一个MyKeys，然后添加自己的快捷输入

以上便是我们常会用到的IDEA快捷键了，如有更多需求，可参考以下或者网上查阅：
