Typora和Markdown语言简单讲解

简介

Typora 是一款 Markdown 编辑器，拥有 Windows、macOS 客户端，其拥有一下几大特点：

  • 自由随心
  • 实时无缝预览
  • 所见即所得

修饰

加粗

** 内容 ** 或者__ 内容 __

快捷键：ctrl+B

效果展示：

  • Typora

  • Typora

斜体

*内容 * 或者_ 内容_

快捷键：ctrl+L

效果展示：

  • Typora
  • Typora

下划线

<u> 内容 </u>

快捷键：ctrl+U

效果展示：

  • Typora

删除线

~~ 内容 ~~

效果展示：

  • Typora

分割线

输入三换以上的*、-、_ 。

效果展示：

版式

标题

  • 一级标题

  • 二级标题

  • 三级标题

  • …

快捷键：ctrl+ 1，2 …

注意：最多只有六级标题

区块

  • 通过符号 >

效果展示：

>

效果展示：

>

目录

输入[TOC]，然后按回车就可以创建目录

效果展示：

[TOC]

代码块

单行代码

把需要添加代码块的部分，用两个 ` 包起来

效果展示：

我的代码

多行注释

把需要添加代码块的部分，用前后```包起来

效果展示：

数学表达式

使用格式：或$$$数学公式$$$

效果展示：

列表

无序列表

在需要的地方使用*、+ 或者 - 加上空格

==注：注意空格==

效果展示：

  • 无序列表
  • 无序列表
  • 无序列表

多级无序列表

在需要用到多级无序列表的内容前面输入多个*、-或+

效果展示：

  • 无序列表

    • 无序列表

      • 无序列表
      • 无序列表

有序列表

在需要用到有序列表的内容前面输入数字.，然后空格。

效果展示：

  1. 列表一
  2. 列表二
  3. 列表三
  4. 。。。

任务列表

使用格式：- [ ] 文字

注：注意空格

效果展示：

  • [x] JAVA
  • [x] Python
  • [ ] C++

表格

一般格式
| 姓名 | 性别 | 年龄 |

快捷键：ctrl+T

效果展示：

姓名性别年龄

注：此处的方式有一些问题

链接

插入图片

插入图片采用格式：! [标题] (路径)

注：路径也可以为网站路径

本地图片路径：右键 > 属性 > 位置 > 将 \换成/，加上图片名称.后缀名

效果展示：

插入链接

插入连接格式：[标题](路径)

效果展示：

百度

注释

可能网站上无法显示这一部分，但在Typora内是可以使用的…

注释文字

使用格式：要添加注释的文字[^注释内容]

效果展示：

中国1

表情的使用

使用格式：:单词:

效果展示：

:smile:

:cry:

Typora快捷键

快捷键作用快捷键作用
Ctrl+1一阶标题Ctrl+B字体加粗
Ctrl+2二阶标题Ctrl+I字体倾斜
Ctrl+3三阶标题Ctrl+U下划线
Ctrl+4四阶标题Ctrl+Home返回Typora顶部
Ctrl+5五阶标题Ctrl+End返回Typora底部
Ctrl+6六阶标题Ctrl+T创建表格
Ctrl+L选中某句话Ctrl+K创建超链接
Ctrl+D选中某个单词Ctrl+F搜索
Ctrl+E选中相同格式的文字Ctrl+H搜索并替换
Alt+Shift+5删除线Ctrl+Shift+I插入图片
