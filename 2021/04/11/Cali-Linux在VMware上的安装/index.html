Cali_Linux在VMware上的安装

Cali_Linux在VMware上的安装

这是一个关于黑客系统的虚拟机安装，我是一直很爱黑客这个的，所以对于大部分黑客喜欢使用的这个系统Cali也是去了解了下，居然刚刚学了虚拟机，这里就是用虚拟机安装一个系统

环境准备

  • VMware 15.5.0

下载地址：https://mp.weixin.qq.com/s/t3ahG9qP1ezQuxap50ncdg

  • Cali Linux

下载地址：https://www.kali.org/downloads/（镜像文件）

https://www.offensive-security.com/kali-linux-vm-vmware-virtualbox-image-download/（虚拟机压缩文件）

下载好这两个软件并且安装好VMware后就可以进行下面的步骤了

压缩包安装方式

在这个页面下载好压缩包

然后进行安装就好了把文件打开就好了

备注：

1、下载的压缩包默认的账号密码为kali/kali

2、cali创建时一般是使用的普通用户，如果需要使用超级管理员root，则需要设置一下

设置root的密码

设置好了后就可以登录root了

镜像安装方式

1、在官网下载Cali系统

2、创建一个虚拟机，选择典型安装

3、选择镜像文件位置

4、选择客服及操作系统为Linux，版本为Ubuntu

5、选择系统安装的位置

6、设置磁盘的大小，设置储存文件为单个文件

7、基本设置完成

8、打开虚拟机配置设置，更改网络适配器为桥接

9、打印机没什么用，就直接移除了

10、启动虚拟机，选择第一个Graphical install

11、选择安装的语言

12、然后等待安装

13、设置主机名

14、设置用户名和密码

15、磁盘分区就选择默认的就好（小白专用）

16、将文件都放在一个分区

17、然后结束分区设置，进行安装

18、选择安装的软件（GNOME和KDE和Large三选一）

  • 这里注意要选择三个中的一个，其余的就选择性安装就好了

19、GRUB启动引擎选择是

20、选择位置为/dev/sda，然后等待安装完成

21、如此便是安装完成了

22、进入桌面看看
