这是我的一个简单的学习的前端的部分的知识，主要学习了uni-app这个简单的前端框架，uni-app是一个使用vue.js开发的前端应用的框架，主要用于开发小程序

知识准备：HTML，CSS，JS，一定量的Vue

1.uni-app的项目创建

脚手架搭建项目

在准备创建的项目的文件夹下面Shift+右键，打开powerShell

  • 全局安装

运行命令：

npm install -g @vue/cli

对此也可下载组件提示语法：

npm i @dcloudio/uni-helper-json

注意：如果已经安装，就不需要再次安装了

  • 创建项目

运行命令：

vue create -p dcloudio/uni-preset-vue my-project

如果出现下列问题：

vue : 无法加载文件 F:\nodejs\node_global\vue.ps1，因为在此系统上禁止运行脚本。

……

使用命令：

Set-ExecutionPolicy -Scope CurrentUser

打开设置，设置值为：

RemoteSigned

  • 启动项目

运行命令：

cd [文件夹名]

npm run dev:mp-weixin（注意是这个命令，不是serve）

  • 微信小程序开发者工具导入项目

选择目录：

示例：

E:\web前端学习\uni-app项目\my-project\dist\dev\mp-weixin

2.基本语法

1.数据展示：

  • 在js中的 data 中定义数据
  • 在 tempalte 中通过 {{数据}} 来显示
  • 在标签中通过 :data-index= '数据' 来使用

2.循环语法

  • 通过 v-for 来指定要循环的数组
  • item 和 index 分别为 循环项 和 循环索引
  • :key 指定唯一的属性，用来提高循环效率

3.条件编译

  • 通过v-if来决定显示和隐藏，不适合做频繁的切换显示

通过在显示时直接删除标签来实现标签的隐藏（标签在工具内无法找到）

  • 通过v-show来决定显示和隐藏，适合做频繁的切换显示

通过样式的更改来实现标签的隐藏（标签可以在工具内被看到，但不显示）

4.计算属性

  • 可以理解为是对data中的数据提供的一种加工或者过滤的能力
  • 通过computed来定义计算属性
